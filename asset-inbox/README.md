# Asset Inbox

Drop source images, spritesheets, audio, and other raw game assets here when you want the agent to import them into the generated project.

Suggested flow:

1. Put files in this folder.
2. Tell the agent which file to use.
3. The agent can optimize or rename it, copy it into `public/assets/`, and update `src/game/assets/assetManifest.ts`.

This folder is a working inbox, not the final runtime asset location.
